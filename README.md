## Description
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
```bash
$ npm install
```
## Running the app
# development
```bash
$ npm run start $
```

# watch mode
```bash 
$ npm run start:dev $
```

# production mode
```bash 
$ npm run start:prod $
```
